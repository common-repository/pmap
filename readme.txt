== Description ==

    Map plugin adds google map to the admin panel of the site and allows you to display it on the page
    Use shortcode [pmap] and attributes width, height to change pam options [pmap width="" height=""]
    

== Installation ==

Upload the pMap plugin to your blog, Activate it, use it.
